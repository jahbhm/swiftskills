🟡 Intermediate 🟣 Coding Challenge Q7

Refactor the function below to use async/await instead of a completion handler. The function fetches a user from a remote API. Make sure error handling is preserved.

struct User: Decodable {
    let id: Int
    let name: String
}

enum APIError: Error {
    case invalidURL
    case decodingFailed
    case networkError(Error)
}

func fetchUser(id: Int, completion: @escaping (Result<User, APIError>) -> Void) {
    guard let url = URL(string: "https://api.example.com/users/\(id)") else {
        completion(.failure(.invalidURL))
        return
    }
    URLSession.shared.dataTask(with: url) { data, _, error in
        if let error = error {
            completion(.failure(.networkError(error)))
            return
        }
        guard let data = data,
              let user = try? JSONDecoder().decode(User.self, from: data) else {
            completion(.failure(.decodingFailed))
            return
        }
        completion(.success(user))
    }.resume()
}

Write the refactored async function. You may use throws instead of Result if you prefer.

🔴 Advanced 🔵 Code Reading Q9

Will this code compile? If yes, does it behave correctly at runtime? If there is a problem, identify it precisely and explain how to fix it.

import Foundation

actor ImageCache {
    private var cache: [String: Data] = [:]

    func store(_ data: Data, for key: String) {
        cache[key] = data
    }

    func retrieve(for key: String) -> Data? {
        return cache[key]
    }
}

class ImageLoader {
    let cache = ImageCache()

    func loadImage(url: URL) {
        URLSession.shared.dataTask(with: url) { data, _, _ in
            guard let data = data else { return }
            self.cache.store(data, for: url.absoluteString)  // Line A
            DispatchQueue.main.async {
                let cached = self.cache.retrieve(for: url.absoluteString)  // Line B
                print("Got image: \(cached != nil)")
            }
        }.resume()
    }
}

Focus on Line A and Line B. Does this code compile? Does it run correctly? Why or why not?
